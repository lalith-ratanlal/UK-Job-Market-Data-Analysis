# UK Job Market Data Analysis

## Project Overview
This project performs exploratory data analysis on a UK job advertisements dataset to uncover hiring trends, salary distributions, role patterns, and regional demand.

## Key Objectives
- Clean and standardize salary data using regex and numerical extraction
- Analyze job posting trends by date and day-of-week
- Perform NLP-based bigram analysis on job titles
- Generate word clouds for role-based insights
- Visualize hiring concentration across UK counties
- Build interactive geospatial choropleth maps

## Techniques Used
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Natural Language Processing (NLTK)
- Geospatial Data Visualization (GeoPandas)
- Interactive Visualizations (Plotly)

## Tools & Libraries
Python, Pandas, Seaborn, Matplotlib, NLTK, GeoPandas, Plotly

## Insights Generated
- Identified peak hiring days and seasonal patterns
- Highlighted high-demand technical roles
- Mapped regional hiring concentration across the UK
